# Available Tools

## task_create
Create a new task or to-do item for the user.
Trigger: user wants to remember something, add a task, set a reminder.

## research
Search the web for current information using Perplexity.
Trigger: user asks about news, facts, or anything requiring up-to-date information.

## recall
Search long-term memory for relevant facts about the user.
Trigger: user references past conversations, preferences, or personal details.

## conversation
General reasoning and conversation.
Trigger: everything else.

## code_generate
Generate, explain, or debug code.
Trigger: user shares code or asks about programming.
